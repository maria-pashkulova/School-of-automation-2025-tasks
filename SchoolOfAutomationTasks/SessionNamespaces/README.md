## About
This project contains solutions for Session_1_Namespaces programming exercises:
* Create a custom namespace called HomeworkUtilities with a class Printer that prints a message.
* Declare variables of different types and use keywords like const, var, and static.
* Perform arithmetic and logical operations on two integers and print the results.
* Bonus Challenge (Optional) 
Create a calculator class inside a namespace SimpleCalculator with methods for basic arithmetic operations.
